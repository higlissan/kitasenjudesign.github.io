## Asics Graphic Generator

* Format / Desktop App 
* Client / Asics
* Role / 実装(Unity)
* URL / --

「全部カラダなんだ」というアシックスのタグラインのもと、ポスター・ウェブ・映像に使用するグラフィック生成ツールを作りました。
スポーツ選手・歌手・ダンサーなどのモデルに合わせ、本人をモーションキャプチャーしたデータの手足をブラシに見立ててグラフィックを生成します。
